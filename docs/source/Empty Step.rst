Empty Step
==========

The Empty Step is a pipeline step that does nothing to the data. It is useful as a placeholder.


.. code-block:: python

    DSPipeline.ds_pipeline.EmptyStep(self)

Parameters
----------

**None**

-------

+---------------+----------------+-----------------+

Example
-------

.. code-block:: python

    import numpy as np
    import pandas as pd
    from DSPipeline.ds_pipeline import EmptyStep, Pipeline

    X = pd.DataFrame(np.random.uniform(size=(10, 4)), columns=['x1', 'x2', 'x3', 'x4'])
    y = pd.Series(np.random.uniform(size=10), name='y')

    empty_step = EmptyStep()
    new_X, new_y = empty_step.fit(X, y)
